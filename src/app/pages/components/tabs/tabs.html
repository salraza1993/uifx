A flexible and accessible tab component for organizing content into separate views. Switch between different sections with a simple click or keyboard navigation.

Pattern 2: Data-Driven with @for Loop

Dynamic tabs generated from an array. Perfect for tabs driven by API data or signals.

@for (tab of tabs(); track tab.value) { {{ tab.title }} } @for (tab of tabs(); track tab.value) {

{{ tab.content }}

 }
Pattern 3: Rich Content with Custom Templates

Tabs with complex content like avatars, badges, icons. Supports any HTML in tab headers and panels.

 }
