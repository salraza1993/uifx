Button Component

Create interactive and visually appealing buttons with our customizable Button component.
Enhance user engagement with a variety of styles , sizes , and states to suit any application need.

Basic

Text to display on a button is defined with the label property.

Directive

Button can also be used as directive using uifxButton along with uifxButtonLabel helper directives.

Link / Router Link

A button can be rendered as a link when type="link" or severity="link" property is present, while the uifxButton directive can be applied on an anchor element to style the link as a button.

Button with Icon

Buttons can include icons using as props iconStart and iconEnd or can be rendered as icon-only buttons using the iconOnly property.

Also, icons can be added using helper directives uifxButtonIconStart and uifxButtonIconEnd or by using ng-templates / any tag with the same names. can be added both side as well.

Loading State

Busy state is controlled with the [loading]="booleanValue" property.

Disabled

When disabled is present, the element cannot be edited and focused.

Severity

Severity defines the type of button. Property as severity="value"

Available severity values are primary secondary danger success warning info light dark gray text

Rounded

Rounded buttons have a circular border radius as property [rounded]="booleanValue" .

Text

Text buttons are displayed as textual elements. Property as [variant]="text"

Outlined

Outlined buttons are displayed as outlined elements. Property as [variant]="outlined"

Icon Only

Buttons can have icons without labels. [iconOnly]="true" and [icon]="iconName"

Badge

Buttons have built-in badge support with [badge]="numberValue" property.

Sizes

Button provides 'ex-small' | 'small' | 'large' | 'ex-large' sizes as alternatives to the standard. Property as size="ex-small"

Button Group

Multiple buttons are grouped when wrapped inside an element with ButtonGroup component. 'ex-small' | 'small' | 'large' | 'ex-large' sizes as alternatives to the standard. Property as size="ex-small"
